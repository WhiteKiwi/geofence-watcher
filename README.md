# geofence-watcher

`geofence-watcher` detects when tracked entities move into or out of configured geofences and can run actions in response.

## Scripts

- `pnpm build`
- `pnpm dev`
- `pnpm lint`
- `pnpm typecheck`

## Commands

- `geofence-watcher tracked-entities ...`
- `geofence-watcher tracked-entity-states ...`
- `geofence-watcher locations ...`
- `geofence-watcher geofences ...`
- `geofence-watcher rules ...`
- `geofence-watcher actions ...`
- `geofence-watcher watch`
- `geofence-watcher install-cron --every 5m`
